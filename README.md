# mpv_fruit

# 配置特性：

0.基于mpv player/mpv_lazy重新组装（Reassemble based on mpv player/mpv_lazy.）。

1.将mpv_lazy项目下部分配置、其他.lua脚本重新组装到mpv player。

2.osc_plus.lua改名为osc_fruit重新排版并修改样式，其他.lua脚本内的字符串改为中文并修改部分样式。

3.组装了anime4K着色器。

4.组装了四款补帧脚本。

5.预设了四款配置供不同性能平台使用。

6.重新定义了快捷键并绘制快捷键说明书。
